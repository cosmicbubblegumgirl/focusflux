# FocusFlux

FocusFlux is a standalone static MVP for an adaptive focus system for ADHD brains.

It includes:

- Local account creation and login
- Browser-persisted Fluxbase data storage
- Focus Pulse check-in
- Flux Fracture task slicing
- Drift recovery with re-entry prompts
- Co-Current body-double room signals
- Pulse Tether fidget input
- Spark Trail progress rewards
- Fluxprint session learning
- Adaptive Web Audio focus cues

The live GitHub Pages version stores data in `localStorage` on the user's device. For production multi-device accounts, connect the same UI to a hosted auth and database service.
